This mod adds a range of interoperability features with mods I like, and is intended to be an addon for Hex Casting first and foremost.
I took a lot of shortcuts to get it done right now (Ack! I couldn't figure out how to set up kotlin, even!), so the code is not in any kind of shape and is something I will fix up later, likely whenever Hexdummy is up to date (for dummies like me).
<details>

<summary>Some of the features this addon includes are</summary>
- Crystal Fishing Rod renamed to Crystal Casting Rod and cost changed to Charged Amethyst

- Crystal Casting Rod can open the hexcasting grid

- Amethyst Bobber cost changed to Charged Amethyst

- Amethyst Bobber now executes stored Hex on reel (so cool...)

- Unlucky Bait which prevents fish from ever biting

- Loud Line which prevents certain mobs from attacking you when line is cast

- Hookless Hook which prevents your thrown bobber from getting caught on entities

- Tacklebox Chair which automatically buckets, or otherwise takes a seated player's recently caught fish into its own inventory. (It's nice!)

- Message in a Bottle left by fishers long gone. Contains some random junk and some random tresaure! It is quite rare and won't be the easiest way to progress, but at least it will be an option. Found in warm ocean biomes.

- Blessed Bobber which adds a very low chance (<0.1%) to reel a random catch from a combined pool of all fish, crates and loot at equal distribution

- Blessed Bobber acts the same as the Amethyst Bobber now does otherwise

- Royalty-Free Music Disc added, "Returning To The Surface" by "cookies+policy" on freesound.org

- Fish added (half-implemented)

- Another casting rod as an upgrade to the Crystal Casting Rod, without durability
